# Regalo Project

## Overview
The Regalo project is a heartfelt web application designed to provide uplifting messages and memories to a loved one. It serves as a digital gift, offering comfort and joy through interactive elements and beautiful design.

## Project Structure
```
Regalo
├── index.html        # Main entry point for the web application
├── css
│   └── styles.css    # Styles for the web application
├── js
│   └── app.js        # JavaScript code for interactivity
├── data
│   └── messages.json  # JSON file containing messages
├── .gitignore        # Files and directories to ignore in Git
└── README.md         # Documentation for the project
```

## Features
- **Interactive Messages**: Users can click a button to receive random uplifting messages.
- **Responsive Design**: The application is designed to look great on both desktop and mobile devices.
- **Memory Gallery**: Users can view a gallery of shared memories through images.

## Setup Instructions
1. **Clone the Repository**:
   ```bash
   git clone <repository-url>
   cd Regalo
   ```

2. **Open the Project**:
   Open `index.html` in your web browser to view the application.

3. **Modify Messages**:
   To update the messages displayed, edit the `data/messages.json` file.

## Usage
- Click the "Presionar" button to receive a random message that aims to uplift and encourage.
- Explore the gallery of memories to reminisce about special moments.
